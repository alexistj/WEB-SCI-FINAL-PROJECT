Arrays

Before you start, let go over a couple of data structures starting with arrays. You can think of an Array as a collection of bins that can hold items of the same data type. When creating an array, you give it a size and a data type. The amount of items cannot go over the defined array size AND all items have to be the same data type. You can access and changeitems by using indecies of the array to locate the location of the items. Arrays don't know their own size so you have to keep track of that your self.

Stacks and Queues

Stacks are a container of objects that remove(pop) and insert(push) objects to the top of the container using the last-in first-out (LIFO) principle

Queues are the opposite in the sense that they follow the first-in first-out princple meaning the first item place in the container is removed but push works the same adding objects to the top of the container.
